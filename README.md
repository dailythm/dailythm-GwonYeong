# dailythm-GwonYeong

## 목적

-   파이썬을 이용해 스택, 큐 등의 자료구조를 배우고 깊이 우선 탐색, 너비 우선 탐색 등 알고리즘 공부

## 파이썬을 선택한 이유

-   문법이 간결함.
    -   문자열을 다루기가 쉬움.
    -   배열 또한 다루기 쉬움.
    -   메소드 등이 직관적임.
